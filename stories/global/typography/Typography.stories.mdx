<br />

#### Importação de fontes

As fontes são importadas em um arquivo css em sua aplicação, por padrão a aplicação trás a
font Nunito Sans que pode ser encontrada no [google-fonts](https://fonts.google.com/ 'ir para google fonts')

_styles/fonts.css_

```css
@import url('https://fonts.googleapis.com/css2?family=Nunito+Sans:ital,wght@0,200;0,300;0,400;0,600;0,700;0,800;0,900;1,200;1,300;1,400;1,600;1,700;1,800;1,900&display=swap');
```

Toda fonte da biblioteca usa a variável _--font-family_ que esta presente no pseudo elemento :root.

```css
:root {
	--font-family: 'Nunito Sans', sans-serif;
}
```

Existe algumas variações para o tamanho, peso, altura de uma fonte, que tambem estão dentro do pseudo :root, são elas:

```css
:root {
	--font-family: 'Nunito Sans', sans-serif;
	--font-size-60: 60px;
	--font-size-48: 48px;
	--font-size-34: 34px;
	--font-size-24: 24px;
	--font-size-18: 18px;
	--font-size-16: 16px;
	--font-size-14: 14px;
	--font-size-12: 12px;

	--line-height-72: 72px;
	--line-height-56: 56px;
	--line-height-40: 40px;
	--line-height-32: 32px;
	--line-height-24: 24px;
	--line-height-16: 16px;

	--font-weight-bold-x: 800;
	--font-weight-bold-b: 700;
	--font-weight-bold-s: 600;
	--font-weight-regular: 400;
}
```

### Predefiniçãoes de textos:

Exite alguns textos ja predefinidos para auxiliar no desenvolvimento.
Você pode adicionar os textos a predefinição atraves da propriedade _class_

##### Predefiniçãoes disponiveis

-   _title-1_
-   _title-2_
-   _title-3_
-   _headline_
-   _sub-headline_
-   _text_
-   _text-bold_
-   _text-sm_
-   _text-sm-bold_
-   _text-caption_
-   _text-btn-large_
-   _text-btn-medium_
-   _text-btn-small_
